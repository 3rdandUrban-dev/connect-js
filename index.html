• For best results, use the official Facebook Connect library which provides a very rich solution without any of the hassles mentioned here.
  • This is a set of examples, and not a library. They demonstrate various Facebook functionality that can be integrated on your site without using the official library. These are a lightweight and less feature-full versions of a subset of functionality provided by the official library. Again, use the official library for best results.
  • Pre-specified heights are used based on the current design used by Facebook for the common case. The official library automatically resizes to fit the content and provides a much better user experience.
  • A cross domain communication technique is used to provide a purely client side experience. The web server only provides static files for this example. But the same approach of using redirects can be used to provide a purely server side experience without using any JavaScript at all - or a hybrid of the two.
  • JSONP enabled API call functionality is included to demonstrate the process of signing and making API calls. The official library does not suffer from JSONP's limitations and uses Flash or Cross Frame communication to provide full cross domain request support.
  • What you see is all there is. No XFBML Tags or Widgets here. Those are provided by the official Facebook Connect library.
  • The examples comprise of:
    • the core example code
    • the cross domain receiver file
    • this html page for the console
    • console styles
    • console javascript
    • JSON2
    • a md5 library
